Welcome to out Tutos home

This is the web site for the WebContext Team, trying to provide some Java tutorials, but not on Java language, but about tooling !

The first tutorial will be a Maven one.

Just discover how to use and implement a pom.xml in your pjava project , to manage version, packaging and release for your project, like profesional people :)

$ git clone git@github.com:Web-Context/tutorial-maven.git

then just do a :

$ mvn clean install

And observe the samples projects, reading the maven-stepX(*.html file to discover and try to understand the magical capabilities of the Maven tools.
